## Welcome to GovTech Multicloud Platform

A production-ready infrastructure platform that allows organizations to migrate between cloud providers (AWS, OCI, GCP, Azure) without rewriting code. Built with interface-based abstraction and factory pattern for true cloud-agnostic architecture.

**Live Demo:** [http://3.237.42.230](http://3.237.42.230)

## Technology Stack

<AccordionGroup>
  <Accordion title="Backend" icon="server">
    - **Runtime**: Node.js 20
    - **Framework**: Express 4.22
    - **Database**: PostgreSQL 15
    - **Cloud SDKs**: AWS SDK, OCI SDK, GCP SDK (planned)
    - **Patterns**: Strategy Pattern, Factory Pattern, Dependency Injection
  </Accordion>

  <Accordion title="Frontend" icon="browser">
    - **Library**: React 18
    - **Build Tool**: Vite 5
    - **Styling**: TailwindCSS 3.4
    - **Routing**: React Router 6
    - **HTTP Client**: Axios
  </Accordion>

  <Accordion title="Infrastructure" icon="cubes">
    - **Containers**: Docker + Kubernetes (EKS)
    - **IaC**: Terraform 5.0
    - **CI/CD**: GitHub Actions + OIDC
    - **Security**: WAF, GuardDuty, Security Hub, CloudTrail, KMS
    - **Monitoring**: CloudWatch Container Insights + Prometheus + Grafana
  </Accordion>

  <Accordion title="Database" icon="database">
    - **Engine**: PostgreSQL 15
    - **Development**: postgres:15-alpine container (port 5432)
    - **Production**: AWS RDS Multi-AZ with automated backups
    - **Connection Pooling**: Built-in with health monitoring
  </Accordion>
</AccordionGroup>
